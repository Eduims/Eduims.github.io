getWorkdays()

lib.php : 18

Count the number of working days between two dates.

getWorkdays(string $date1, string $date2[, bool $workSat = FALSE ][, string $patron = NULL ]) : int

This Function calculate the number of working days between two given dates, taking account of the Public festivities, Easter and Easter Morning days, the day of the Patron Saint (if any) and the working Saturday.

Parameters
$date1 : string

Start date ('YYYY-MM-DD' format)

$date2 : string

Ending date ('YYYY-MM-DD' format)

$workSat : bool = FALSE

TRUE if Saturday is a working day

$patron : string = NULL

Day of the Patron Saint ('MM-DD' format)

Tags
author

Massimo Simonini massiws@gmail.com

Return values
int —

Number of working days ('zero' on error)
